Calculate π by Leibniz formula. Short formula:
```
1 - 1/3 + 1/5 - 1/7 + 1/9 - ... =  π/4
```

See: [wikipedia](https://en.wikipedia.org/wiki/Leibniz_formula_for_%CF%80)
